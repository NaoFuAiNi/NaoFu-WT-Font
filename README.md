# NaoFu WT Hub 2.10

本工具用于在本地制作并替换《战争雷霆》相关字体资源，仅供学习与个人娱乐使用。

---

## 运行环境

- **系统**：Windows 10 或 Windows 11
- **使用方式**：解压后即可使用，无需安装。双击 **NaoFu WT Hub.exe** 启动
- **依赖**：发布包为自包含，无需单独安装 .NET；界面依赖系统自带的 WebView2 运行时（Windows 10/11 通常已内置，若提示缺失请另行安装）

## 使用方法

1. **启动**：双击运行 **NaoFu WT Hub.exe**（或运行 bin 目录内的 NaoFu WT Hub 2.10.exe）
2. **选择字体**：在界面中选择或拖入你的字体文件（支持 .ttf / .otf）
3. **开始制作**：输入项目名与作者名，选择模式后点击「开始制作」
4. **输出**：制作完成后可在「当前选择」中查看自制字体，使用「打开生成字体的目录」打包发给好友；或直接将某字体「导入到游戏 UI 文件夹」

## 目录说明（发布包）

- **bin**：主程序与运行所需文件，请勿删除
- **tools**：字体处理相关程序，请勿删除
- **ui / assets / font**：界面与字体资源（font 下含 source、builtin、custom），请勿删除
- **build**：制作完成后的输出会出现在此目录下
- **font/custom**：可将自定义字体放入此处（或通过界面选择）

# NaoFu's NaoFu WT Customize Font 1.10
## 战争雷霆自定义字体替换工具

### 项目简介

《战争雷霆》玩家此前替换游戏内字体比较麻烦。本工具通过二进制定位与替换，在保持官方 `fonts.vromfs.bin` 结构的前提下，把用户指定字体写进对应位置，替换后即可在游戏内生效（HUD、聊天等）。

### 使用方法（1.10 时期）

1. 准备你的字体文件，复制到 `custom_font`，重命名为 `MyFonts.ttf`
2. 双击运行 `NaoFu_WT_Font_Patcher.exe`，按序号选择要替换的字体
3. 完成后将 `build` 下的 `fonts.vromfs.MODDED.bin` 复制到游戏 `ui` 目录替换原文件
4. 启动游戏即可
